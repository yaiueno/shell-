# レポート作成に向けて不足しているもの

`mondai.txt` と実行ログ、ソースコードを照合した結果、提出用レポートに入れる前に確認・修正した方がよい項目です。

## 要修正のソースコード

- 問題2: `2/1.sh` は `ABC = A` のように代入の前後に空白があり、ログでも `ABC: not found` が出ている。`ABC=A; A=BC` に直す必要がある。
- 問題4: `4.sh` の5行目が `ecoh` になっている。`echo` に修正する必要がある。
- 問題11: `owner.sh` は現在 `w`, `r`, `x` の順に表示している。課題例に合わせるなら `r`, `w`, `x` の順にする。
- 問題12: `12/conf12.sh` は `i = $((i+1))` のように代入の前後に空白がある。`i=0` の初期化も追加する。
- 問題14: `14/14.sh` は `sum = ...` や `j = ...` に空白があり、1行の中でコマンド区切りも不足している。
- 問題24: `24/getgid.sh` は `グループ名=GID` の順で出力している。課題例に合わせるなら `GID=グループ名` にする。

## 実行ログが不足しているもの

- 問題2: `2/2.sh` は引数なしで実行されて算術式エラーになっている。例として `./2.sh 8 12` などで再実行する。
- 問題7: `sub.sh`, `mul.sh`, `div.sh`, `res.sh` の実行ログが見当たらない。
- 問題8: `8.sh` の実行ログが見当たらない。
- 問題10: `calc.sh` の実行ログが見当たらない。`+`, `-`, `*`, `/` は最低限確認する。
- 問題11: `owner.sh` の実行ログが見当たらない。対象ファイルを用意して実行する。
- 問題12: `12/p12.sh`, `12/conf12.sh` の実行ログが見当たらない。
- 問題23: `23/23.sh` は対象となる `bar*.tex` がないため、置換前後を確認できるテストファイルが必要。
- 問題24: `24/getgid.sh` は実行ログはあるが、課題例と列順が逆なので修正後に再実行する。
- 実習6: `j6.sh` は作成済み。`./j6.sh 5` や `./j6.sh 10` で実行し、出力例をレポートへ追記する。

## レポート本文でまだ本人入力が必要なもの

- 表紙の学生番号、名前、実習実施日、提出日。
- 実習3と実習6〜9のうち、実際に授業で扱った範囲。現状のディレクトリだけでは内容を完全には特定できないため、作業中のものは空欄のまま残している。
- 参考文献欄。授業資料名や配布PDF名を記入する。
